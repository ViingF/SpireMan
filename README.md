SpireMan项目说明



一、项目简介

本项目是一个使用 C++20 和 SFML3.0.2 开发的卡牌冒险游戏。游戏包含主菜单、角色选择、地图探索、战斗、奖励、商店、营火、事件和结算等流程。大部分数值大小都和杀戮尖塔1中一致，也有改编，如考虑到游戏难度，将敌人血量下调了70%



二、运行方式

可执行文件位于 release 目录下。

运行时请保持 exe 文件和所有 dll 文件放在同一个目录中，否则程序可能因为缺少运行库而无法启动。



>双击 release 目录下的 exe 文件即可运行程序





三、目录说明

assets

存放游戏运行所需的资源文件，例如图片、背景、角色立绘、怪物图片、按钮图片、图标、音乐和音效等。程序启动和切换场景时会从该目录读取资源。

data

存放游戏数据文件，例如卡牌数据、敌人数据、遭遇数据、事件数据等。程序通过这些数据生成战斗内容、地图内容、奖励内容和事件内容。

logs

存放运行日志。程序运行时会记录启动、加载资源、读取数据、保存进度、场景切换和错误信息等内容，便于调试和排查问题。

src

存放项目源代码。



四、src 源代码结构说明

src/main.cpp

程序入口，负责初始化日志系统，创建游戏对象并启动主循环。

src/config

存放项目配置和常量，例如资源路径、窗口大小、地图层数、商店价格、营火回复比例、敌人数值调整参数等。

src/core

存放游戏核心框架代码，包括游戏主循环、资源管理、音频管理、日志系统、场景基类和场景切换逻辑。

src/database

存放数据读取和数据库管理代码，用于读取卡牌、敌人、遭遇和事件等外部数据，并提供给游戏系统使用。

src/model

存放游戏中的基础数据结构，例如玩家、敌人、卡牌、牌组、地图节点、事件、遭遇和当前游戏进度等。

src/scene

存放各个游戏场景的实现，包括主菜单、角色选择、地图、战斗、奖励、商店、营火、删卡、事件和结算场景。

src/system

存放主要游戏逻辑系统，包括卡牌效果、战斗流程、敌人 AI、地图生成、事件处理和存档读写等。

src/ui

存放界面组件代码，包括按钮、卡牌显示、血条、能量显示、敌人意图、顶部信息栏、奖励卡牌、商店界面和状态显示等。
